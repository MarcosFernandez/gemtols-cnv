CNV PIPELINE¶

Copy number pipeline¶

Performs copy number calls based on Read Depth corrected by GC content in genome windows. Read Depth is calculated from Whole Genome Sequencing (WGS) data aligned to a masked (Simple and Tandem repeats) reference genome.

Resources¶

CNV GEMTOOLS PIPELINE¶

Set of tools to perform all the pipeline.

cnv gemtools pipeline

JIP GEM CNV PIPELINE¶

Use JIP pipeline system to run all the pipeline in three commands. (1. Assembly, 2. Pipeline Configuration, 3. Pipeline execution).

jip cnv gemtools pipeline
